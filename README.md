# StartupHotfix

## About

Fixes issues with Valheim's 0.221.10 update by setting the correct DistributionPlatform earlier.
Steam and Xbox are supported.
This is a patcher and goes into the patchers folder!

## Manual Installation

This mod requires [BepInEx](https://thunderstore.io/c/valheim/p/denikson/BepInExPack_Valheim/).
Extract the content of `StartupHotfix` into the `BepInEx/patchers` folder or any subfolder.
